# Multifunction Calculator

A feature-rich desktop calculator developed with **Visual Basic .NET**, **Windows Forms**, and **Microsoft Visual Studio 2010**. The application combines everyday calculations with scientific, engineering, financial, and utility tools into a single easy-to-use desktop application.

Designed as an all-in-one calculation suite, it provides solutions for mathematical operations, scientific calculations, engineering formulas, financial computations, unit conversions, and age calculations.

---

## Features

### Basic Calculator
- Addition
- Subtraction
- Multiplication
- Division
- Percentage calculations
- Decimal operations
- Clear and reset functions

### Scientific Calculator
- Trigonometric functions (Sin, Cos, Tan)
- Square and square root
- Exponents and powers
- Logarithmic functions
- Natural logarithms
- Factorial calculations
- Pi (π) and other mathematical constants

### Boyle's Law Calculator
Calculate unknown gas properties using **Boyle's Law**.

Formula:

> P₁V₁ = P₂V₂

Users can calculate:
- Final Pressure
- Initial Pressure
- Final Volume
- Initial Volume

### Age Calculator
Quickly calculate:
- Current age
- Years
- Months
- Days
- Total elapsed time between two dates

### Unit Converter
Convert between multiple measurement units, including:
- Length
- Weight/Mass
- Temperature
- Volume
- Area
- Time

### Loan Calculator
Estimate loan repayments based on:
- Loan amount
- Loan duration (months)
- Monthly repayment
- Total repayment

## Getting Started

### Prerequisites

- Microsoft Visual Studio 2010 or later
- .NET Framework 4.0 (or the version targeted by the project)

### Installation

1. Clone the repository.

```bash
git clone https://github.com/aloskhalifa/multifunction-calculator.git
```

2. Open the `.sln` file in Microsoft Visual Studio.

3. Build the solution.

4. Run the application.
